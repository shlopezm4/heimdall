## Heimdall
Spring Boot Application + JWT + MySQL 


<b>Built with</b>
- [Spring Boot](http://spring.io/projects/spring-boot)


## How to use?
If you want to sign up you have to use the next URL pattern:
```
***/api/auth/signup
```
If you want to sign in you have to use the next URL pattern:
```
***/api/auth/signin
```
